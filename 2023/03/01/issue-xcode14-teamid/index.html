Xcode 14 build 報錯：套件沒有 teamID

Winnie
2023-03-01
React Native, Xcode

Xcode 14 在 build app 時會遇到「套件沒有 teamID」這個錯誤而導致 build 失敗。在 XCode 裡可以在介面點擊錯誤，手動新增 teamID，但這樣效率低。可以在 Podfile 裡新增一段 script，暫時解決這個問題。

這個 issue 官方已經解決了。
